Uninstalling
============

You can manually remove the install and build directory.

.. tabs::

    .. group-tab:: Linux

        .. code-block:: bash

            rm -rf cyclonedds
            rm -rf <install-location>

    .. group-tab:: macOS

        .. code-block:: bash

            rm -rf cyclonedds
            rm -rf <install-location>

    .. group-tab:: Windows

        Navigate to your install location and remove the |var-project| folder.
